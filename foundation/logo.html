Logo Variations

Greyscale

The grayscale logo should be used when printing in color is not an option.

Color

The color logo is the primary logo and sohuld be used against light backgrounds.

Color Inversed

The color inversed logo should be used against dark backgrounds to ensure legibility.

Greyscale Inversed

The Greyscale inversed logo should be used against dark backgrounds to ensure legibility.
