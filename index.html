This is not a drill:
Design system

 

About

 

"This is not a drill" is a design system developed with inspiration and influences from 90's 8bit graphic theme. The theme evolved to adopt usability guidelines and improve the readability by using more user friendly readable primary typeface.

This system includes three sections Foundations, Components, Page Patterns. covering typographical hierarchy, color palette, and logo variations in Foundations. Components for navigation bars, footers, and cards in Components. Three views of a homepage and a case study page in page patters
