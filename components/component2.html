Footer

Example

©Copyright

Code

          
              <Footer class="footer-sai py-1 fixed-bottom">
              ©Copyright
             </Footer>

Usage

This footer shall be used in all the pages of the website at the bottom using sticky bottom command.

The must include the copyright information and the other information maybe optional.

Accessibilty Considerations

This footer passes all WCAG Compliances including Small Text, Large Text and UI Components with the contrast ratio of 9.09:1. If the footer is being used in the lighter backgrounds a different shade of grey(Grey 1 or Grey2) maybe used.
